# Experiment 1: Basic Java Programs

## Aim
To implement basic Java programs using control structures and arrays.

## Programs Included
1. Factorial of a number  
2. Prime number check  
3. Sorting an array  
4. Merge two arrays  
5. Matrix operations (addition, multiplication, transpose)

## Tools Used
- Java  
- JDK  
- VS Code  
- Command Line  

## Learning Outcome
Learned array operations, loops, condition checking, and use of command line arguments in Java programs.
