# Experiment 10: Java GUI using Event Handling

## Aim
To implement different types of event handling using Java Swing components.

## Programs Included
1. GUI program to display current mouse coordinates on the window using MouseMotionListener
2. GUI program to implement a simple Timer with Start and Stop buttons using background events
3. GUI with JComboBox to select image names and display the selected image using JLabel and ItemListener
4. GUI with JTextArea to display character count and word count in real-time using KeyListener
5. GUI program to change background color using JColorChooser

## Tools Used
- Java
- Swing
- IntelliJ IDEA / VS Code
- JDK

## Learning Outcome
- Learned event handling in Java
- Understood different listeners like MouseMotionListener, ActionListener, ItemListener, and KeyListener
- Implemented real-time GUI updates
- Learned how to handle user interactions in applications
- Gained practical knowledge of building interactive GUI programs
