# Experiment 12: Java GUI using Layout Managers

## Aim
To implement different types of layout managers using Java Swing components.

## Programs Included
1. BorderLayout: GUI program to convert a number into Binary, Octal, and Hex using BorderLayout
2. FlowLayout: Java program using FlowLayout (left aligned, horizontal gap 10px, vertical gap 20px) with checkboxes "Java", "Python", and "C++"
3. GridLayout: Java program using GridLayout (2x3) to display numbers 1 to 6 and swap numbers on button click
4. GUI program to find factorial of a given number using Applet

## Tools Used
- Java
- Swing
- IntelliJ IDEA / VS Code
- JDK

## Learning Outcome
- Learned different layout managers in Java Swing
- Understood BorderLayout, FlowLayout, and GridLayout
- Implemented GUI applications using layouts
- Learned how to arrange components properly in a window
- Gained practical knowledge of designing structured user interfaces
