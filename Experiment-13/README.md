# Experiment 13: JDBC Connectivity and GUI Application

## Aim
To implement different database operations using JDBC.

---

## Programs Included

### 1. Java-MySQL Database Connectivity
Java program to establish connection between Java and MySQL database and display contents of the student table.

### 2. CRUD Operations using JDBC
Java-MySQL database connectivity program to perform following operations on the student table:
- Select
- Insert
- Update
- Delete

### 3. GUI Login Form using Swing
GUI based Java program to create a login form with username and password fields and store registration data in MySQL database.

---

## Tools Used
- Java
- JDBC
- MySQL
- Swing
- VS Code / IntelliJ IDEA
- JDK
- MySQL Connector Driver

---

## Learning Outcome
- Learned how to connect Java with MySQL database using JDBC
- Understood database connectivity concepts
- Performed CRUD operations using SQL queries in Java
- Learned execution of Select, Insert, Update, and Delete operations
- Implemented GUI applications using Java Swing
- Gained practical knowledge of database-driven applications
- Improved understanding of user input handling and database interaction
