# Experiment no 5: Implementation of User Defined Packages in Java

## Aim
To implement user-defined packages in Java and understand how to organize classes into reusable modules.

## Problems Implemented

1. Library Management Package
Created a package LibraryManagement with classes Book and Member.
- Book class contains attributes: title, author, ISBN
- Member class stores member details
Used this package to create a simple library system.

2. Ecommerce Package
Created a package Ecommerce with classes Product, Customer, and Order.
- Implemented methods for placing an order
- Displaying product details
- Calculating total order cost
This package is reused in another program.

3. Math Operations Package
Created a package MathOperations that includes functions:
- floor()
- ceil()
- round()
Implemented a program to perform mathematical operations using these functions.

## Tools Used

- Java
- JDK
- VS Code / IntelliJ IDEA
- Command Line

## Learning Outcome

- Learned how to create and use user-defined packages in Java
- Understood modular programming and code reusability
- Implemented real-world systems like library and ecommerce
- Gained knowledge of built-in mathematical functions
- Improved program organization using packages

## Conclusion

This experiment helped in understanding how Java packages improve code structure, maintainability, and reusability by grouping related classes together.
