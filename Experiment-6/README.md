# Experiment 5: Java I/O Streams and GUI using Swing

## Aim

To implement various input/output operations using Java I/O Streams and to develop basic GUI applications using Swing.

## Programs Included

1. Reading data from a file (sample.txt) using File and Scanner  
2. Writing and reading student data using DataOutputStream and DataInputStream  
3. File analysis program to count vowels, words, and occurrences of character 'a'  
4. GUI Counter application (Count Up, Count Down, Reset)  
5. GUI program to reverse a number  
6. GUI program demonstrating radio buttons (Gender Selection)  

## Tools Used

- Java  
- JDK  
- VS Code / Notepad  
- Command Line  

## Learning Outcome

- Learned file handling using Java I/O Streams  
- Understood byte stream classes like FileInputStream and FileOutputStream  
- Implemented DataInputStream and DataOutputStream for structured data  
- Developed basic GUI applications using Swing  
- Learned event handling using ActionListener  
- Understood usage of Swing components like JButton, JTextField, JLabel, and JRadioButton
