# Experiment 7: Java I/O Streams

## Aim
To implement various input/output operations using Java I/O Streams.

## Programs Included
1. Java program to read data from a sample.txt file located outside the program directory
2. Java program to accept student details (name, age, weight, height, city, phone number), store them using DataOutputStream and FileOutputStream, and retrieve them using DataInputStream and FileInputStream
3. Java program to read a text file and compute:
   - Total number of vowels
   - Total number of words
   - Number of times character 'a' appears

## Tools Used
- Java
- JDK
- IntelliJ IDEA / VS Code
- Command Line

## Learning Outcome
- Learned file handling using Java I/O Streams
- Understood use of FileInputStream and FileOutputStream
- Learned DataInputStream and DataOutputStream for structured data
- Implemented reading and writing operations in files
- Learned how to process text data (count vowels, words, characters)
- Gained practical knowledge of input/output operations in Java
