# Experiment 9: GUI using Swing Components and IDE

## Aim
To design GUI applications using Java Swing components and implement them in IntelliJ IDEA.

## Programs Included

1. Standard Calculator using Swing components  
   - Performs basic arithmetic operations such as addition, subtraction, multiplication, and division  
   - Uses `JTextField` for input/output  
   - Uses `JButton` for digits and operations  
   - Displays the result in the text field  

2. Stationary Purchase System  
   - Allows the user to select stationary items such as Notebook, Pen, and Pencil  
   - Takes quantity as input using dialog boxes  
   - Calculates total amount based on selected items  
   - Displays order confirmation and total bill  

## Tools Used
- Java
- Swing
- IntelliJ IDEA
- JDK

## Learning Outcome
- Learned how to create GUI applications using Java Swing
- Understood the use of `JFrame`, `JTextField`, `JButton`, `JCheckBox`, and `JOptionPane`
- Implemented event handling using `ActionListener`
- Learned how to perform arithmetic operations through GUI
- Understood how to take user input through dialog boxes
- Learned how to display messages and results in GUI applications
- Gained practical knowledge of developing simple desktop-based applications in Java

## Description

### 1. Standard Calculator
This program is designed using Swing components to create a simple calculator interface. It supports the four basic mathematical operations:
- Addition
- Subtraction
- Multiplication
- Division

The calculator uses:
- `JTextField` to show numbers and results
- `JButton` for digits and arithmetic operators
- Event handling for button click actions

This application helps in understanding GUI design, user interaction, and implementation of arithmetic logic in Java.

### 2. Stationary Purchase System
This program simulates a small stationary ordering system. The user can:
- Select items using checkboxes
- Enter quantity for selected items
- View total bill amount
- Receive confirmation message after placing the order

The program uses:
- `JCheckBox` for item selection
- `JButton` for placing order
- `JOptionPane` for input and output dialogs

This application demonstrates the practical use of Swing components for creating form-based interactive applications.

## Conclusion
This experiment helped in understanding how Java Swing can be used to design interactive GUI applications. It also improved knowledge of event handling, input/output display, and implementing real-life mini applications using Java.
